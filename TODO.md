# Workforce Monitoring and Insider Risk Management Agent

A comprehensive system for monitoring employee productivity, ensuring compliance, and detecting insider risks using Linux endpoint monitoring and AI-powered analysis.

Implement user activity tracking, data loss prevention (DLP), and behavioral analytics to enhance productivity. 
IMplement time tracking, application usage.


Real-time and recorded session monitoring to observe user actions and ensure productivity.


Data Loss Prevention (DLP): track and prevent the unauthorized transfer of sensitive data.
Track and prevent data exfiltration.

User Behavior Analytics (UBA) / User Entity Behavior Analysis (UEBA): AI-driven analytics to identify suspicious or risky user behavior and anomalies.

 **Productivity Analysis**: Calculates productivity metrics and identifies optimization opportunities
    - Productivity metrics calculation, Productivity score calculation.
   - Insider risk assessment. Insider risk score calculation.
   - LLM-powered behavioral analysis
   - Pattern recognition and anomaly detection

Reporting and Auditing: Detailed logs and reports for investigations, compliance, and productivity analysis.


Interactive frontend with charts and real-time updates
Real-time alerts for suspicious activities

Agent in C++
Backend Server in Python
